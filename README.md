# Bilibili-Video-Downloader

这是一个哔哩哔哩视频下载器，提供以下特性：

## 主要功能

- 支持自定义视频清晰度选择
- 按视频原始标题命名下载文件
- 支持DASH格式下载
- 分别下载视频和音频流
- 使用FFmpeg快速合并音视频
- 自动清理临时文件
- 检查FFmpeg可用性

## 使用要求

- Python 3.7+
- FFmpeg
- requests库

## 安装依赖

pip install requests
使用方法
安装FFmpeg
Windows: 下载并配置系统PATH
macOS: brew install ffmpeg
Linux: sudo apt-get install ffmpeg
